WOOD CUSTOM CORNERS THEME

Only suitable for Moodle 1.9 upwards!

This theme is one working proof of concept of two amazing CSS features built within Moodle:

  • Inheriantce between themes: This theme has defined the standard "wood" theme as its parent so, automatically, all the styles/modifications/improvements from that theme will be available and displayed here.
  • Custom corners: Since Moodle 1.9 is really easy to build new themes whose main boxes (header, footer, blocks, contents...) are enclosed by rounded boxes, generating a cool effect.

This theme is based on Moodle's Standard, Wood and Custom Corners 2 themes.

To work properly, both Moodle Standard and Wood themes must be present and unaltered in the "theme" directory.

In this way custom themes are not "left behind" by Moodle upgrades.

For any suggestion, fix or comment about this theme, please, start a discussion in the "Themes Forum".
